# SubFlow — Project Context

## Identitas
Nama         : SubFlow
Tipe         : Desktop app Windows — AI subtitle pipeline
Stack        : Go (Wails v2) + Svelte + SQLite + faster-whisper
Target OS    : Windows 10+ 64-bit dan Linux Linux (macOS untuk masa depan)
Bahasa UI    : Bahasa Indonesia
Bahasa Kode  : English (semua variable, function, comment)

## Apa yang Dibangun
Aplikasi desktop yang otomasi pipeline subtitle end-to-end:
  Import (video/subtitle/URL)
  → ASR (faster-whisper lokal / Groq / Deepgram)
  → Source Correction (LLM)
  → Context Reconstruction (metadata scene)
  → Segmentation (42 char/baris, 2 baris, 17 CPS)
  → Translation Layer 1 (DeepL/OpenAI/Anthropic/Gemini/Ollama)
  → AI Rewrite Layer 2 (6 providers + 4 tone presets)
  → QA Engine (9 checks otomatis + auto-fix)
  → Export (SRT/VTT/ASS/TXT, single/Dual Subtitle)

## Hardware ASR yang Didukung
CPU          : selalu tersedia, int8 quantization
NVIDIA CUDA  : deteksi via nvidia-smi, CUDA >= 11.8, float16
AMD ROCm     : deteksi via rocm-smi, ROCm >= 5.0, float16
Apple CoreML : darwin/arm64 saja (macOS future build)
Intel OpenVINO: deteksi path OpenVINO runtime

## Pipeline State (urutan wajib)
imported → audio_extracted → asr_done → corrected → context_done
→ segmented → translated → rewritten → qa_done → exported

## Prinsip Non-Negotiable
1. Konten user tidak pernah lewat server SubFlow
2. Tidak ada dependency runtime yang harus diinstall user
3. Installer ≤ 200MB termasuk model Whisper base
4. Semua error punya kode ERR_[MODULE]_[NUMBER]
5. Pipeline resumable setelah crash

## Data Directory
Windows : %APPDATA%\SubFlow\
macOS   : ~/Library/Application Support/SubFlow/
Linux   : ~/.local/share/subflow/

## Project State Files (per project)
{data_dir}/projects/{project-id}/
  raw_asr.json, corrected.json, context.json,
  segmented.json, translated.json, rewritten.json,
  glossary.json, qa_report.json

## Error Code Format
ERR_IMP_* → import   ERR_ASR_* → ASR
ERR_TRN_* → translation   ERR_RWT_* → rewrite
ERR_QA_*  → QA   ERR_EXP_* → export
ERR_DB_*  → database   ERR_SYS_* → system
